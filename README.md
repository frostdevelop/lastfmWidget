# lastfmWidget
A cool little widget for Twitch streamers!
## Prerequisites
- You need a last.fm account to use this. 
- For compatibility on streaming services other than Spotify, you must use a browser extension (I recommend [this](https://webscrobbler.com/))
## Instructions
- Download `lastfmWidget-<version>-min.html` from the latest release.
- Double click the downloaded file to open it
- Create a last.fm account [here](https://www.last.fm/join)
- Get an API key [here](https://www.last.fm/api/account/create) (Fields in this form are optional)
- After creating your API key, it will take around 5 minutes for it to register in their system.
- Enter your Last.fm account username and API key, change the settings to your liking and copy the url at the bottom.
- Insert that url as a browser source in your preferred streaming software.
- The browser source's dimensions should exactly the dimensions of your stream. (e.g. 1280x720 or 1920x1080)
